#  Proyecto Programación III – Sistema de Juegos y Perfiles

Aplicación de escritorio desarrollada en **Java** como parte de la materia **Programación III / Ingeniería de Requerimientos**.  
El objetivo del proyecto es gestionar **perfiles de usuarios** y **módulos de juegos**, utilizando programación orientada a objetos y formularios con Swing.

---

##  Objetivos del proyecto

- Practicar los principios de **POO** (clases, objetos, herencia, composición, etc.).
- Implementar una estructura de proyecto ordenada para una aplicación de escritorio en Java.
- Gestionar **perfiles de jugadores** y **servicios de juegos** desde una misma interfaz.
- Trabajar con **formularios**, validaciones y lógica de negocio separada del código de la UI.

---

##  Funcionalidades principales

> *(La lista se puede ir actualizando a medida que avances)*

- Registro, edición y eliminación de **perfiles de usuario**.
- Listado de perfiles registrados.
- Gestión de **módulos de juegos** (crear, actualizar, listar).
- Asociación de juegos a perfiles.
- Formularios gráficos en **Swing** (ventanas, botones, campos de texto).
- Manejo básico de validaciones y mensajes al usuario.

---

##  Tecnologías utilizadas

- **Lenguaje:** Java  
- **Paradigma:** Programación Orientada a Objetos  
- **Interfaz gráfica:** Swing  
- **IDE:** IntelliJ IDEA  
- **Control de versiones:** Git y GitHub
